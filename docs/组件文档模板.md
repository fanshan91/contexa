# 组件文档模板

> 复制本模板并按组件实际情况补全（建议与 Storybook stories 同步维护）。

## 组件名

**文件位置**
- `apps/web/components/ui/<component>.tsx` 或 `apps/web/components/common/<component>.tsx`

**用途**
- 说明组件解决的问题与典型使用场景。

## Props

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| className | string | - | 追加样式（通过 `cn()` 合并） |
| ... | ... | ... | ... |

## 示例

```tsx
import { Button } from "@/components/ui/button";

export function Demo() {
  return <Button>Demo</Button>;
}
```

## 可访问性

- 键盘导航：说明 Tab/Enter/Escape 等关键交互是否支持。
- ARIA：需要 `aria-label` 的场景与要求。

## 设计规范

- Tokens：使用的语义化类名（如 `bg-background`、`text-muted-foreground` 等）。
- 变体：`variant`/`size` 支持范围（如有）。

## 注意事项

- 是否为 Client Component（是否依赖 Radix/hook）。
- 常见误用与边界条件。
