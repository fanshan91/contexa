# 组件清单（apps/web）

本文用于汇总当前 `apps/web` 可复用组件，方便在页面开发时优先复用与扩展，避免重复实现。

## 1. 原子 UI（components/ui）

目录：[components/ui](file:///Users/nihao/Desktop/project/products/TMS/contexa/apps/web/components/ui)

- `avatar.tsx`：Avatar
- `button.tsx`：Button（cva variants）
- `card.tsx`：Card
- `dialog.tsx`：Dialog（含 Header/Footer 等子组件）
- `dropdown-menu.tsx`：DropdownMenu（含 Item/Group/Sub 等）
- `input.tsx`：Input
- `label.tsx`：Label
- `radio-group.tsx`：RadioGroup
- `toast.tsx`：Toast（含 Provider/Viewport 等）

## 2. 通用组件（components）

目录：[components](file:///Users/nihao/Desktop/project/products/TMS/contexa/apps/web/components)

- `device-guard.tsx`：设备/环境保护与提示
- `form-error.tsx`：表单错误展示
- `page-error.tsx`：页面错误展示
- `language-switcher.tsx`：语言切换
- `locale-select.tsx`：语言选择
- `brand/logo-mark.tsx`：品牌 Logo

## 3. 待补齐（来自 UI通用组件封装和优化.md）

优先以 `components/ui`（shadcn 风格）补齐：

- 表单基础：Form、Select、Checkbox、Switch、Textarea
- 数据展示：Table、Badge、Tabs、Separator、Skeleton、ScrollArea
- 导航与反馈：Breadcrumb、Sheet、Tooltip、Alert、Progress、Collapsible
- 业务级通用（建议放在 `components/common`）：DataTable、PageTree、StatusBadge
